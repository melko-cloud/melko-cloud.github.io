Set a login method for a user

path Parameters
login_method
required
string (LoginMethod)
Enum: "magic-link" "sms-otp" "password"
Request Body schema: application/json
required
user_id
required
string
password
string

The password to set. Only required when login_method is password.

must_update_password
boolean
Default: false

Whether the user must update their password on their next login.

notify_user
boolean
Default: false

Sent the user a copy of the password. Only applicable when login_method is password.

notification_url
string <uri>

The URL to send the user to in the notification email. Only applicable when notify_user is true.
